# Microsoft Teams Cache Cleaner

🇺🇸 Automated Windows batch script to clear Microsoft Teams cache and restart the application.  
🇧🇷 Script automatizado em batch para limpeza de cache do Microsoft Teams e reinicialização do aplicativo.

---

## 🇺🇸 Overview

This script was created to optimize IT support troubleshooting by automating the Microsoft Teams cache cleanup process.

It performs the following actions:

- Terminates running Teams processes
- Removes local cache files
- Restarts Microsoft Teams (Store and Classic versions supported)

---

## 🇧🇷 Visão Geral

Este script foi criado para otimizar o troubleshooting no suporte de TI, automatizando o processo de limpeza de cache do Microsoft Teams.

Ele realiza as seguintes ações:

- Finaliza processos do Teams em execução
- Remove arquivos de cache locais
- Reinicia o Microsoft Teams (versão Store e Clássica)
